# **App Name**: LexCompage

## Core Features:

- Home Page: Showcase the advocate's name, tagline, and a professional image with practice areas and a 'Book Consultation' button.
- About Page: Display the advocate's biography, qualifications, practice areas (Civil, Criminal, Constitutional Law), jurisdiction information (Jharkhand High Court & Civil Courts), and an admin-managed photo gallery of the office.
- Contact Form: Allow users to submit a contact form. The form is sent to the advocate via EmailJS, and includes validation.
- Updates/Case Information: Enable the admin to create and display blog-style posts about notable cases.
- Admin Dashboard: Secure admin login (Supabase Auth) to manage case updates, upload images, manage office photos, and view contact form submissions.
- Office Photo Upload and Management: Allow the admin to upload, reorder, and delete office photos.
- Contact Form Submission Management: Display a list of contact form submissions within the admin panel.

## Style Guidelines:

- Primary color: Navy Blue (#1e3a5f) to convey authority and trust.
- Accent color: Gold (#d4af37) to convey prestige and professionalism.
- Background color: Off-white (#f9f9f9) to maintain a clean, professional appearance.
- Headline font: 'Playfair', serif for headings, provides a formal, authoritative feel.
- Body font: 'Inter', sans-serif, for body text, ensuring readability and a clean interface.
- Use classic, simple icons to represent key actions and practice areas.
- Maintain a clean, mobile-first layout with collapsible navigation and large touch targets.
